# Installation Guide

## Quick Start

1. **Navigate to project directory:**
   ```bash
   cd d:\Projects\SetGalaxy\setgalaxy-clone
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   ```
   http://localhost:3000
   ```

## Alternative Installation

Run the setup script:
```bash
setup.bat
```

## Build for Production

```bash
npm run build
npm start
```
